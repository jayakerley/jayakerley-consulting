# Jay Akerley Consulting

Static multi-page website for Cloudflare Pages.

## Files

- `index.html` — homepage
- `garden.html` — The Garden
- `consulting.html` — Consulting
- `speaking.html` — Speaking
- `about.html` — About Jay
- `journal.html` — Journal landing page
- `contact.html` — Contact page
- `styles.css` — shared styling
- `logo-rabbitbrush.svg` — rabbitbrush logo

## Images

Images are in `/images` and use the filenames selected for the site.

## Deploying to Cloudflare Pages

1. Push these files to the GitHub repository.
2. In Cloudflare Pages, connect the repository.
3. Use these build settings:
   - Framework preset: `None`
   - Build command: leave blank
   - Output folder: `/` or the repository root
4. Deploy.
